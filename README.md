📊 Interactive Stock Market Dashboard for the Indian IT Sector
About the Project

This project is an interactive business intelligence dashboard designed to analyse and visualise stock market performance within India's IT sector. The dashboard focuses on three key datasets: Tata Consultancy Services (TCS), Infosys, and the NIFTY IT Index, enabling users to explore historical trends, trading activity, and future stock price forecasts through an intuitive web-based interface.

Built using Python, Dash, Plotly, Pandas, and Prophet, the dashboard combines descriptive and predictive analytics to support data-driven decision-making for investors, analysts, researchers, and students.

Key Features
📈 Interactive stock price trend analysis
📊 Trading volume visualisation
🔥 Correlation heatmaps for market insights
📅 Dynamic date-range filtering
🏢 Company selection (TCS, Infosys, NIFTY IT Index)
🤖 Prophet-based stock price forecasting
📋 Automated descriptive statistics
💾 Downloadable filtered datasets
🖱️ Interactive charts with hover tooltips

- Business Value

The dashboard transforms raw stock market data into actionable insights by allowing users to monitor historical performance, compare companies against sector benchmarks, identify trading patterns, and forecast future trends. It demonstrates how data visualisation and predictive analytics can support investment decisions and financial market analysis.

Dataset

The project uses historical stock market data for:
- Tata Consultancy Services (TCS)
- Infosys
- NIFTY IT Index

covering daily trading metrics including Open, High, Low, Close, and Volume.
